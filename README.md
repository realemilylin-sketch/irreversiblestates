# Irreversible States

**Irreversible States** is a reference framework defining irreversible conditions and finality thresholds in real-world processes.

This repository is a **public reference mirror** of the documentation site:
- Primary reference: https://irreversiblestates.com

## What this is
- A conceptual framework and vocabulary for describing **finality points**:
  - acceptance / handoff finality
  - responsibility transfer
  - loss of optionality
  - irreversible thresholds in processes

## What this is not
- Not legal advice
- Not a product or service
- Not a normative standard (reference only)
